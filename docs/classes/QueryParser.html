    Class QueryParser

    QueryParser parses OData query strings into structured query objects. It handles $select, $filter, $orderby, $expand, $skip, and $top query options.

    Example

    const queryParser = new QueryParser(
      '/users?$select=name,email&$filter=age gt 18&$orderby=name asc&$top=10',
      dataSource,
      User,
      { defaultTop: 100, defaultSkip: 0 }
    );

    const results = await dataSource.execute(queryParser);
