    Class DataSource

    DataSource manages the database connection and model registration. It serves as the central point for database operations and entity management.

    Example

    const dataSource = new DataSource({
      database: 'mydb',
      username: 'user',
      password: 'pass',
      host: 'localhost',
      dialect: 'postgres',
      port: 5432,
      pool: { max: 5, min: 0, idle: 10000 },
      schema: 'public',
      models: [User, Order, Product]
    });
